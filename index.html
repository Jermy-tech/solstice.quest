How to Join

Follow these simple steps to join Solstice SMP

1

Install Minecraft

Option 1: Official Minecraft (Recommended)

  1. Go to minecraft.net and purchase Minecraft Java Edition
  2. Download the Minecraft Launcher and install it
  3. Launch the game, log in, and proceed to Step 2

Option 2: TLauncher (Alternative Free Option)

  1. Download TLauncher from tlauncher.org
  2. Install TLauncher and open it
  3. Click Accounts > Manage Accounts > Create an Account and enter your username
  4. Select Minecraft Version 1.21.4, then click Install & Play
2

Connect to the Server

Java Edition:

IP: play.solstice.quest

Bedrock Edition:

IP: nil

Port: 1825

  1. Open Minecraft (or TLauncher)
  2. Click Multiplayer > Add Server
  3. Enter the IP and click Done
  4. Join the server and start your adventure!
3

(Optional) Enable Proximity Voice Chat

Want to talk with nearby players? Install Simple Voice Chat mod for PaperMC:

Download Here

How to Install the Plugin (Client-Side)

  1. Download the .jar file from the link above
  2. Install Fabric or Forge (TLauncher users can do this in the settings)
  3. Place the .jar file in your Minecraft mods folder:
  4. Windows: Press Win + R, type %appdata%, and navigate to .minecraft/mods/
  5. Mac: Open ~/Library/Application Support/minecraft/mods/
  6. Linux: Go to ~/.minecraft/mods/
  7. Launch the game using Fabric/Forge
  8. Once in-game, press V to configure your voice settings
4

Enjoy the Server!

If you have any issues, ask for help in #support channel on our Discord.
